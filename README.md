# Personal configuration V2

This is my personal configuration for Neovim with Lua and some awesome plugins. With this version it is possible to use several languages, if is necessary to add other languages just modify the mason and tree-sitter files.


## LSP

This version support several programming languages, such as:

- Lua
- Latex
- Java
- Javascript
- Typescript
- Go
- Rust
- Zig
